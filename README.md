# 🏥 Documentación Oficial - FRO Salud SpA

**Proyecto:** Desarrollo de una solución de software integral con el propósito de centralizar los servicios y la interacción usuario-profesional en FRO Salud SpA.

**Asignatura:** Ingeniería de Software I  

**Año:** 2026  

---

## 📖 Descripción del Repositorio
Este repositorio contiene la documentación técnica, arquitectónica y de gestión correspondiente al desarrollo del sistema **FRO Salud**. Los artefactos aquí alojados respaldan las decisiones de diseño, el levantamiento de requerimientos y la planificación ágil (Scrum) del proyecto.

### 📌 Estado Actual: Incremento 1
En este primer incremento, la documentación se centra en los módulos fundacionales del sistema:
- Gestión de Usuarios y Seguridad.
- Agendamiento y Citas.
- Arquitectura base y Base de Datos.

---

## 📂 Estructura del Repositorio

El repositorio está organizado de la siguiente manera para facilitar su revisión:

- 📄 **`Entrega del Incremento 1.docx`**: Documento principal del incremento con el informe completo.
- 📄 **`Grupo 17 - Documento 0.docx`**: Documento base del proyecto.
- 📁 **`/Anexo`**: Carpeta que contiene los diagramas y modelos en **alta resolución** para su correcta inspección técnica. Incluye:
  - Modelo Entidad-Relación Extendido (MERE).
  - Modelo Físico de Base de Datos.
  - Árbol de Navegación de las Interfaces.
